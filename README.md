# TaskFlow API Documentation

This repository contains structured REST API documentation for a task management system.

It demonstrates professional API documentation structure including authentication, endpoints, error handling, and version tracking.

---

## Project Overview

TaskFlow API allows developers to:

- Create tasks
- Update tasks
- Retrieve task lists
- Handle authentication securely
- Understand structured error responses

## Documentation Structure
## Postman Collection

You can test the API using the exported Postman collection:

1. Download the collection from the `postman/` folder.
2. Open Postman.
3. Click Import.
4. Select the JSON file.
5. Run requests directly.

This collection includes:
- GET /tasks
- POST /tasks
- PUT /tasks/{id}

- API-Reference.md → Complete endpoint definitions
- Authentication.md → Token-based authentication guide
- Getting-Started.md → First request walkthrough
- Errors.md → Error response handling
- Release-Notes.md → Version history and updates
