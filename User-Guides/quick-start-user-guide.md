# Quick Start Guide: Taskflow

## Overview
Taskflow is a task management platform that helps individuals and teams create, assign, and track tasks in one place. This guide helps new users get started with the Taskflow web application.

## Who this guide is for
This guide is for users who are new to Taskflow and want to create and manage tasks using the dashboard. No technical knowledge is required.

## Step 1: Create an account
1. Open the Taskflow website.
2. Click **Sign Up**.
3. Enter your email address and password.
4. Click **Create Account**.

After signing up, you are redirected to the Taskflow dashboard.

## Step 2: Create a task
1. From the dashboard, click **New Task**.
2. Enter a task title.
3. (Optional) Add a task description.
4. Select a due date.
5. Click **Save**.

The task is added to your task list.

## Step 3: Assign a task
1. Open the task you want to assign.
2. Click **Assign User**.
3. Select a team member from the list.
4. Click **Assign**.

The assigned user can now view and work on the task.

## Step 4: Track task progress
Each task has a status that shows its progress:
- **To Do** — Task has not started
- **In Progress** — Task is being worked on
- **Completed** — Task is finished

You can update the task status from the task details page.
