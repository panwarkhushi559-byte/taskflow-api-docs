# Release Notes

This document tracks changes and improvements made to the TaskFlow API documentation.

---

## v1.1.0 – Improvements and Enhancements

Release Date: January 2026

### Added
- Task filtering by priority
- Pagination parameters (page and limit)
- Improved error response examples

### Updated
- Authentication documentation clarified
- Getting Started guide expanded

### Fixed
- Minor formatting issues in API reference tables

---

## v1.0.0 – Initial Release

Release Date: January 2026

### Features
- Create Task endpoint (POST /tasks)
- Update Task endpoint (PUT /tasks/{task_id})
- List Tasks endpoint (GET /tasks)
- Token-based authentication
- Structured error responses

This was the first stable version of the TaskFlow API documentation.
