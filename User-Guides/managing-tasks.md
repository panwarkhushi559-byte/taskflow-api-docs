# Managing Tasks in Taskflow

## Overview
This guide explains how to create, edit, assign, and manage tasks in Taskflow using the web dashboard.

## Create a task
1. From the dashboard, click **New Task**.
2. Enter a task title.
3. (Optional) Add a task description.
4. Select a due date.
5. Click **Save**.

The task is added to your task list.

---

## Edit a task
1. Open the task you want to edit.
2. Click **Edit**.
3. Update the task details such as title, description, or due date.
4. Click **Save** to apply the changes.

---

## Assign a task
1. Open the task you want to assign.
2. Click **Assign User**.
3. Select a team member from the list.
4. Click **Assign**.

The assigned user can now view and work on the task.

---

## Change task status
Each task has a status that shows its progress:
- **To Do** — Task has not started
- **In Progress** — Task is being worked on
- **Completed** — Task is finished

To update the status:
1. Open the task.
2. Select the new status from the status menu.

---

## Delete a task
1. Open the task you want to delete.
2. Click **Delete**.
3. Confirm the action.

Deleted tasks cannot be recovered.

---

## Common issues

### Task is not visible
- Check if the task is assigned to you.
- Make sure the correct project is selected.

### Cannot assign a user
- Ensure the user is part of the same team.
- Check your account permissions.
